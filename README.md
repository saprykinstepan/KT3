# Программа анализа кусочно-заданных функций
## Описание:
Курсовая работа по дисциплине "Основы программирования и алгоритмизации". 
Программа на языке C для комплексного анализа кусочно-заданной математической функции.
## Функциональность
- **Вычисление значения функции в любой точке**
- Табулирование функции на заданном интервале (вывод на экран и сохранение в файл)
- Анализ монотонности функции на отрезке
- Поиск корня уравнения f(x) = Y методом бисекции
- Численное дифференцирование (вычисление производной)
## Основные функции программы

### `f(double x)`
Вычисляет значение кусочно-заданной функции по заданным математическим формулам.

### `build_table(double start, double step, int count)`
Формирует и выводит на экран таблицу значений функции на указанном интервале.

### `save_table_to_file(double start, double step, int count, const char* filename)`
Сохраняет таблицу значений функции в текстовый файл без вывода на экран.

### `check_monotonic(double start, double end, double step)`
Анализирует характер изменения функции на заданном интервале (возрастание/убывание).

### `find_x(double Y, double tolerance, double start, double end)`
Решает уравнение f(x) = Y методом бисекции (половинного деления) на указанном интервале.

### `derivative_at_point(double x, double h)`
Вычисляет приближенное значение производной функции в заданной точке методом правой разности.
