# Программа анализа кусочно-заданных функций
## Описание:
Курсовая работа по дисциплине "Основы программирования и алгоритмизации". 
Программа на языке C для комплексного анализа кусочно-заданной математической функции.
## Функциональность
- Вычисление значения функции в любой точке
- Табулирование функции на заданном интервале (вывод на экран и сохранение в файл)
- Анализ монотонности функции на отрезке
- Поиск корня уравнения f(x) = Y методом бисекции
- Численное дифференцирование (вычисление производной)
